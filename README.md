# HexoBlog_Next
基于Hexo的博客系统，可以本地和服务器联调，主题的经典好看的Next,三种主题方案可选！
博客学习来自以下链接，博客主题来源自Github。
也算是自我产出的开始把，通过服务器建立这样一个静态博客，毕业后终于开始了计算机编程的学习，很有意义的博客！
### 因为是纯静态的，所以不会受到服务器推送的各种被攻击提示，如果单纯当作博客来使用，完全足够了，速度非常快！
